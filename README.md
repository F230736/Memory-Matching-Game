🧠 Memory Matching Game

A simple and interactive **Memory Matching Game** built using **HTML, CSS, JavaScript, and jQuery**. The goal of the game is to match pairs of cards by remembering their positions.

🎮 How the Game Works
- Cards are displayed face-down at the start.
- The player flips two cards at a time.
- If the cards match, they stay flipped.
- If they do not match, they flip back after a short delay.
- The game continues until all pairs are matched.
- A **Restart button** allows the player to start the game again anytime.

✨ Features
- Card flip animation
- Memory-based matching logic
- Restart game option
- Responsive card grid layout
- Simple and clean UI
- Uses emojis/images as card symbols

🛠 Technologies Used
- **HTML** – Structure of the game
- **CSS** – Styling and layout
- **JavaScript** – Game logic
- **jQuery** – Simplified DOM manipulation and event handling
